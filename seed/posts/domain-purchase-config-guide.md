# 域名购买与配置完全指南

域名是网站的门牌号，选择一个好域名并正确配置是建站的第一步。本文将带你走完从域名选购到解析生效的全流程。

## 一、如何选择域名

选择域名时建议遵循以下原则：

1. **简短易记** — 控制在 3-8 个字符最佳，方便用户输入
2. **与品牌相关** — 域名最好包含品牌名或核心关键词
3. **选择合适后缀** — .com 为首选，也可考虑 .cn、.net、.top 等
4. **避免连字符和数字** — 容易混淆，例如 "my-site.com" 不如 "mysite.com"

## 二、域名注册流程

### 2.1 选择注册商

国内常见的域名注册商包括：

- **腾讯云** — 支持 .com/.cn/.top 等常见后缀，与新网合作
- **阿里云** — 万网旗下，国内市场份额领先
- **GoDaddy** — 国际知名注册商，适合海外业务

### 2.2 注册步骤

1. 查询域名是否可用
2. 选择注册年限（建议首次注册 1-3 年）
3. 填写域名所有者信息（个人注册填写身份证信息）
4. 实名认证（国内注册商强制要求）
5. 支付完成

> **注意**：域名注册后需要进行实名认证，通常 1-3 个工作日审核通过，期间域名无法正常解析。

## 三、DNS 配置

### 3.1 什么是 DNS

DNS（域名系统）将人类可读的域名转换为服务器 IP 地址。你可以把它想象成**互联网的电话簿**。

### 3.2 常见 DNS 记录类型

| 记录类型 | 用途 | 示例 |
|---------|------|------|
| A 记录 | 将域名指向 IPv4 地址 | `example.com → 1.2.3.4` |
| CNAME | 将域名指向另一个域名 | `www → example.com` |
| MX 记录 | 邮件服务器配置 | `@ → mail.example.com` |
| TXT 记录 | 文本验证信息 | SPF、DKIM、域名验证等 |
| NS 记录 | 指定 DNS 服务器 | 域名委托给特定 DNS |

### 3.3 配置步骤

1. 登录域名注册商控制台
2. 进入 DNS 管理 / 域名解析页面
3. 添加解析记录：
   - **A 记录**：`@ → 你的服务器 IP`（根域名）
   - **CNAME**：`www → @`（www 子域名）
4. 等待 DNS 生效（通常几分钟到 48 小时）

## 四、常见问题

**Q：域名解析多久生效？**
A：TTL 值决定生效时间。默认 600 秒（10 分钟），修改后需要等待缓存刷新。

**Q：域名过期后会怎样？**
A：过期后进入 30 天赎回期，赎回期后域名释放，他人可注册。建议开启自动续费。

**Q：可以变更域名注册商吗？**
A：可以，通过域名转移功能，但需满足：注册满 60 天、未处于争议期、距到期超 15 天。
